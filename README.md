<h4>💡 기획 의도</h4>
- 공간대여를 중개하는 온라인 플랫폼을 구축 사용자, 공간을 제공자들을 연결하여 쉽고 편리하게 공간을 대여할 수 있는 시스템 
사용자들이 다양한 목적에 맞는 공간을 쉽게 찾고 예약하여, 공간 제공자들이 자신의 공간을 효율적으로 운영하고 수익을 창출 실현
